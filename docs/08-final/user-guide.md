# Руководство пользователя

> Руководство по использованию мобильного приложения TaxiFleet Admin для администратора таксопарка.

---

## 1. Вход в систему

1. Откройте приложение TaxiFleet Admin на мобильном устройстве.
2. На экране входа введите:
   - **Логин:** ваш логин администратора (по умолчанию: `admin`)
   - **Пароль:** ваш пароль (по умолчанию: `admin123`)
3. Нажмите кнопку **«Войти»**.
4. При успешной аутентификации вы будете перенаправлены на главный экран (дашборд).

### Возможные ошибки

| Ошибка | Причина | Решение |
|--------|---------|---------|
| «Неверный логин или пароль» | Некорректные учётные данные | Проверьте правильность логина и пароля |
| «Ошибка подключения к серверу» | Сервер недоступен | Убедитесь, что сервер запущен и устройство имеет сетевое соединение |

---

## 2. Главный экран (Дашборд)

После входа отображается сводная панель со статистикой:

- **Водители:** общее количество и разбивка по статусам (Свободен / Занят / Недоступен)
- **Заказы:** общее количество и разбивка по статусам (Новый / Назначен / В пути / Выполнен / Отменён)
- **Автомобили:** общее количество и разбивка по статусам

Из дашборда доступна навигация к разделам:
- Водители
- Заказы
- Автомобили

---

## 3. Управление водителями

### 3.1 Просмотр списка водителей

1. На дашборде нажмите **«Водители»** или выберите пункт в нижнем меню навигации.
2. Отобразится список всех водителей с цветовой индикацией статуса:
   - Зелёный — Свободен (FREE)
   - Красный — Занят (BUSY)
   - Серый — Недоступен (UNAVAILABLE)

### 3.2 Добавление водителя

1. На экране списка водителей нажмите кнопку **«+»** (добавить).
2. Заполните поля:
   - **ФИО** — полное имя водителя
   - **Телефон** — номер телефона (формат: +7-XXX-XXX-XXXX)
   - **Номер удостоверения** — номер водительского удостоверения
3. Нажмите **«Сохранить»**.
4. Новый водитель появится в списке со статусом «Свободен».

### 3.3 Редактирование водителя

1. Нажмите на карточку водителя в списке.
2. Измените необходимые поля.
3. Нажмите **«Сохранить»**.

### 3.4 Удаление водителя

1. На карточке водителя нажмите иконку **удаления** (корзина).
2. Подтвердите действие в диалоговом окне.

> **Важно:** Нельзя удалить водителя в статусе «Занят» (BUSY). Сначала завершите его текущий заказ.

---

## 4. Управление заказами

### 4.1 Просмотр списка заказов

1. На дашборде нажмите **«Заказы»**.
2. Отобразится список заказов, отсортированный по дате создания (новые сверху).
3. Каждый заказ содержит: имя клиента, маршрут, статус.

### 4.2 Фильтрация заказов

Используйте вкладки в верхней части экрана для фильтрации:
- **Все** — все заказы
- **Новые** — только заказы в статусе NEW
- **В работе** — заказы ASSIGNED и ON_WAY
- **Завершённые** — DONE и CANCELLED

---

## 5. Создание заказа

1. На экране заказов нажмите кнопку **«+ Новый заказ»**.
2. Заполните форму:
   - **Имя клиента** — ФИО пассажира
   - **Телефон клиента** — контактный номер (необязательно)
   - **Адрес отправления** — откуда забрать
   - **Адрес назначения** — куда доставить
3. Нажмите **«Создать»**.
4. Заказ будет создан со статусом **NEW** и появится в списке.

---

## 6. Назначение водителя на заказ

1. В списке заказов нажмите на заказ со статусом **NEW**.
2. На экране деталей заказа нажмите кнопку **«Назначить водителя»**.
3. Из выпадающего списка выберите **свободного водителя** (статус FREE).
4. Из выпадающего списка выберите **доступный автомобиль** (статус AVAILABLE).
5. Нажмите **«Назначить»**.
6. После успешного назначения:
   - Статус заказа изменится на **ASSIGNED**
   - Статус водителя изменится на **BUSY**
   - Статус автомобиля изменится на **ON_TRIP**

> **Примечание:** Если свободных водителей или доступных автомобилей нет, соответствующий список будет пустым.

---

## 7. Управление автомобилями

### 7.1 Просмотр списка автомобилей

1. На дашборде нажмите **«Автомобили»**.
2. Отобразится список всех автомобилей с информацией: марка, модель, госномер, статус.

### 7.2 Добавление автомобиля

1. Нажмите кнопку **«+»** (добавить).
2. Заполните поля: марка, модель, госномер, год выпуска, цвет.
3. Нажмите **«Сохранить»**.

### 7.3 Статусы автомобилей

| Статус | Значение | Цвет |
|--------|----------|------|
| AVAILABLE | Доступен для поездок | Зелёный |
| ON_TRIP | На линии (выполняет заказ) | Синий |
| MAINTENANCE | На техобслуживании | Жёлтый |
| BROKEN | Неисправен | Красный |

---

## 8. Выход из системы

1. Нажмите иконку **выхода** в правом верхнем углу.
2. JWT-токен будет удалён из локального хранилища.
3. Вы будете перенаправлены на экран входа.
